# Project

## How to Run

To start the project, follow these steps:

### 1. Install Dependencies

Open a terminal window and run:

```bash
cd client
npm install
```

Open another terminal window and run:

```bash
cd server
npm install
```

### 2. Start the project

In the client terminal window, run:

```bash
npm run dev
```

In the server terminal window, run:

```bash
npm run start:dev
```
